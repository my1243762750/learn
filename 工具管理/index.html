npm

  • 查看全局安装位置：npm root -g
  • 查看npm当前镜像源: npm config get registry
  • 全局安装yrm：npm i yrm -g

yarn

  • 查看全局安装位置：yarn global dir
  • 查看yarn当前镜像源: yarn config get registry
  • 设置淘宝镜像：yarn config set registry https://registry.npm.taobao.org -g
  • 全局安装yrm：yarn global add yrm

yrm

  • 查看所有资源：yrm ls
  • 使用某个资源：yrm use 资源名称
  • 测试资源速度：yarm test 资源名称

vue

  • 查看webpack所有配置：vue inspect
  • 查看webpack所有规则：vue inspect --rules
  • 查看webpack svg规则：vue inspect --rule svg
