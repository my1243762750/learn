git

  • 设置全局用户名：git config --global user.name 'my1243762750'
  • 设置全局密码： git config --global user.password '***'
  • 设置用户名： git config user.name 'my1243762750'
  • 设置密码： git config user.password 'heromy@13224044'
  • 当前仓库保存用户名密码：git config credential.helper store
  • 全局仓库保存用户名密码：git config --global credential.helper store
  • 去掉github弹框：git config --system --unset credential.helper
  • 查看系统已设置好的配置：git config --list 然后按p退出

常用命令

  • git config http.postBuffer 524288000 (增加缓存空间到500M)
  • git config --system --unset credential.helper (重置用户名，密码)
  • git config --global http.sslVerify false (去掉ssl的验证)
  • git config user.name (查看用户名)
  • git config user.email (查看邮箱)
  • git remote -v (查看项目地址)
  • git branch -a (查看所有分支)
  • git branch -r (查看远程分支)
  • git branch (查看本地分支)
  • git branch 分支名 (创建本地分支)
  • git checkout 分支名 (切换分支)
  • git checkout -b 分支名 (创建+切换分支)
  • git remote show origin (可以看到删除分支情况)
  • git remote prune origin (清除已删除的分支缓存)
  • git push origin 分支名 (将分支推送到github)
  • git branch -d (删除本地分支)
  • git push origin :(分支名) (删除github远程分支)
  • git log (查看日志)
  • 英文状态下 按q (退出查看日志)
  • 合并分支
  • 还原文件
  • 查看历史记录
  • git commit
  • git push
  • git pull

添加ssh过程

  • 找到.ssh 删除它，github上找到“SSH and GPG keys”，删除SSH keys（自行百度）
  • 用git bash执行ssh-keygen -t rsa -C "xxxxx@xxxxx.com" 生成.ssh目录(有id_rsa和id_rsa.pub文件)
  • 把id_rsa.pub里面的内容拷贝到github上（自行百度）
  • 按照以上步骤，不会出现远程无法连接
