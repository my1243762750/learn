js编码和解码的方式

window.location的所有获取值(假如url地址为: http://www.test.com:8081/project/index.html?name='zhangsan'&pwd='123')

  • window.location.href (http://www.test.com:8081/project/index.html?name='zhangsan'&pwd='123')
  • window.location.hostname (www.test.com)
  • window.location.pathname (project/index.html)
  • window.location.port (8081)
  • window.location.protocol (http:)
  • window.location.hash (上面看不出来，再假如url地址为: http://www.test.com:8081/project/index.html#projectUrgent，window.location.hash就是#projectUrgent )
  • window.location.search (?name='zhangsan'&pwd='123')
