git

  • 设置全局用户名：git config --global user.name 'my1243762750'
  • 设置全局密码： git config --global user.password '***'
  • 设置用户名： git config user.name 'my1243762750'
  • 设置密码： git config user.password 'heromy@13224044'
  • 当前仓库保存用户名密码：git config credential.helper store
  • 全局仓库保存用户名密码：git config --global credential.helper store
  • 去掉github弹框：git config --system --unset credential.helper
  • 查看系统已设置好的配置：git config --list
